# fCC Forum Leaderboard

A dynamic web application that fetches and displays the latest topics from the freeCodeCamp forum using their public API. This project was built as part of my web development learning journey.

## 🚀 Live Demo
[View the Live Project Here](https://github.com/Gssaini73/Forum-Leaderboard)

## ✨ Features
- **Real-time Data:** Fetches the most recent forum posts using the `fetch` API and `async/await`.
- **Custom Time Logic:** Calculates and displays how long ago a post was active (e.g., "10m ago", "2h ago").
- **Responsive Design:** A clean, mobile-friendly table layout using CSS Flexbox and Media Queries.
- **Dynamic Category Styling:** Automatically styles post categories with specific colors based on the forum's ID system.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for the leaderboard table.
* **CSS3:** Custom properties (variables) and responsive design.
* **JavaScript (ES6+):** DOM manipulation, Template Literals, and API integration.

## 📖 What I Learned
During this project, I practiced:
1. Handling JSON data from an external API.
2. Mapping through arrays to generate dynamic HTML content.
3. Managing project files by separating concerns into `forum.html`, `forum.css`, and `forum.js`.
